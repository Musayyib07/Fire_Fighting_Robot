# Fire Fighting Robot using Arduino

An autonomous, hardware-based mini-project designed to detect and extinguish localized fires. Built using an **Arduino Uno**, this robot navigates towards a fire source using flame sensors and deploys a localized water pump mechanism to put it out, minimizing human risk in hazardous environments.

## 🚀 Key Features
* **Autonomous Fire Detection:** Utilizes high-sensitivity flame sensors to scan and locate fire sources.
* **Targeted Extinguishing:** Integrates a servo motor to sweep the water nozzle for precise firefighting.
* **Efficient Mobility:** Powered by DC motors and a dedicated L298N/L293D motor driver module for smooth navigation.
* **Automatic Pumping:** Features an on-board 5V/12V DC water pump that activates instantly upon fire confirmation.

## 🛠️ Hardware Stack
* **Microcontroller:** Arduino Uno
* **Sensors:** Flame Sensor Module
* **Actuators:** DC Motors, SG90 Servo Motor, DC Submersible Water Pump
* **Motor Driver:** L298N / L293D Shield
* **Power Source:** Li-ion / Lead-Acid Battery
